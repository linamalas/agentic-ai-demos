# agentic-ai-demos
Agentic AI demos built with LangGraph and Streamlit — featuring multi-agent workflows for code and recipe generation.

## Projects

### Code Generator + Reviewer
- Uses multiple agents for code generation, review, reflection, and improvement.
- Built with LangGraph, Streamlit, and GPT-4o-mini.

### Recipe Generator + Critic
- Two-agent system where one generates recipes and the other critiques them.
- Demonstrates iterative refinement and conversational loops.

## Tech Stack
Python · LangGraph · Streamlit · LangChain OpenAI
